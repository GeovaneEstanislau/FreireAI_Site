Política de
Privacidade

O FreireAI respeita a sua privacidade e está comprometido em proteger seus dados pessoais. Esta política descreve como coletamos e usamos informações através de nossa plataforma e integração com APIs de terceiros.

Uso da API do Google Ads

Nossa plataforma acessa dados da conta do Google Ads apenas com a autorização explícita do usuário via autenticação OAuth 2.0. Utilizamos esses dados exclusivamente para automatizar o gerenciamento de campanhas publicitárias e análise de desempenho.

Nenhum dado é vendido ou compartilhado com terceiros. Todos os processos de autenticação seguem os padrões de segurança OAuth 2.0 do Google.

Coleta de Dados

Coletamos apenas as informações necessárias para operar o serviço, como identificadores de conta para fins de configuração da API e métricas de desempenho de anúncios (CTR, CPC) para alimentar nossos algoritmos de otimização.

Proteção de Dados

Implementamos medidas de segurança técnicas e organizacionais para proteger seus dados contra acesso não autorizado, alteração, divulgação ou destruição.
